# gestion-restaurant
La gestion d’un restaurant repose sur plusieurs éléments clés. Le module Plats (Menu) permet de créer et organiser les recettes avec leurs prix. Les Commandes assurent le suivi des clients. La gestion des Tables optimise l’occupation. Les Réservations planifient l’accueil, tandis que les Employés gèrent les rôles et horaires.
